# Pixel Town Demo - AI小镇游戏项目情况总结

## 项目概述
这是一个简化的2D像素风AI小镇游戏demo，玩家可以控制主角在小镇中移动并与NPC互动。NPC都是具备完整AI能力的Agent，能够进行记忆、反思和计划，并在玩家与之对话后将信息传播给其他NPC。

## 技术栈
- **游戏引擎**: Phaser.js (v3.90.0)
- **AI框架**: LangChain.js (v1.2.32)
- **状态管理**: Zustand (v5.0.12)
- **构建工具**: Vite (v8.0.0)
- **语言**: TypeScript (v5.9.3)

## 当前实现状态
- ✅ 基础游戏框架已搭建
- ✅ 主场景显示背景和玩家角色
- ✅ 玩家可以使用方向键控制角色移动
- ✅ 支持窗口大小调整
- ✅ 资源文件已配置（scene.png, player.png）

## 如何运行游戏
1. 确保已安装依赖：`npm install`
2. 启动开发服务器：`npm run dev`
3. 在浏览器中打开：http://localhost:3000/

## 项目结构
```
src/
├── game/           # Phaser游戏逻辑
├── agents/         # LangChain Agent实现
├── store/          # Zustand状态管理
├── ui/            # React组件
└── utils/         # 通用工具函数
public/
├── assets/        # 像素素材
└── index.html     # 入口HTML
```
