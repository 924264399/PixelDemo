# bug整理


### 3. **JavaScript falsy值陷阱（关键bug）**
- **问题**：`const target = this.targetRoofAlphas.get(key) || 1.0;`
- **原因**：当 `targetRoofAlphas.get(key)` 返回 `0` 时，`0 || 1.0` 返回 `1.0`（因为 `0` 是falsy值）
- **结果**：目标透明度本应是 `0.0`（透明），但实际变成了 `1.0`（不透明）
- **解决方案**：使用空值合并操作符 `??` 或检查 `has()` 方法，或者直接使用sprite的alpha属性避免状态管理



### 关键要点
1. **避免falsy值陷阱**：不要使用 `||` 操作符处理可能为0的数值
2. **简化状态管理**：直接使用sprite的alpha属性，避免维护额外的状态Map
3. **性能考虑**：每帧计算简单，无额外开销
