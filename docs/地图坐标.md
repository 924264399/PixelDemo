# 地图坐标与位置文档

## 地图规格

- **世界尺寸**: 2048 x 2048 像素
- **坐标系统**: 左上角为原点 (0, 0)，X向右增加，Y向下增加
- **碰撞遮罩**: 512 x 512 像素（比例 1:4）

## 坐标转换公式

### 世界坐标 → 遮罩坐标

```typescript
const maskX = Math.floor(worldX / 4);
const maskY = Math.floor(worldY / 4);
```

### 遮罩坐标 → 世界坐标

```typescript
const worldX = maskX * 4;
const worldY = maskY * 4;
```
